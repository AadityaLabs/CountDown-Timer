# CountDown-Timer
The CountDown Timer is a simple and reliable tool that tracks time remaining for tasks, events, and activities. It helps improve productivity, manage deadlines, and maintain focus by providing accurate real-time countdowns with an easy-to-use interface.

# AuraFocus - Premium Study Companion

AuraFocus is a polished browser-based study timer and focus dashboard, built with Tailwind CSS and custom JavaScript. It combines Pomodoro timing, stopwatch mode, task management, ambient 8D audio synthesis, and a premium UI experience.

## Features

- Pomodoro timer with short/long break modes
- Custom countdown mode for personalized focus sessions
- Integrated stopwatch mode
- Task management panel with pin and complete actions
- 8D-style audio engine with binaural focus, rain, and brown noise
- Theme toggle with dark mode support
- Animated ambient particle background and glassmorphism UI
- Toast notifications for key actions
- LocalStorage support for theme and task persistence

## Built With

- HTML
- Tailwind CSS (CDN)
- JavaScript
- Lucide Icons (CDN)
- Google Fonts: Plus Jakarta Sans, Playfair Display

## Project Structure

- `index.html` — main application page
- `style.css` — custom visual styling and effects
- `app.js` — timer engine, audio engine, task manager, and UI logic

## Usage

1. Open `index.html` in a browser.
2. Select a timer mode: Pomodoro, Short Break, Long Break, or Custom.
3. Start/pause the timer using the central controls.
4. Add study tasks and pin the active focus goal.
5. Toggle Zen Mode for a distraction-free overlay.
6. Enable ambient audio with spatial 8D audio controls.

## Notes

- No build tool or installation required.
- The app runs entirely in the browser.
- Local settings and tasks are stored via `localStorage`.
